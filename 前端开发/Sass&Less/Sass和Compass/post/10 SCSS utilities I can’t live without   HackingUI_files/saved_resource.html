          I believe that you have a typo in your "Respond To" that setup does not work, you will need a value or a variable for the conditionals to work against. In your example it's a variable name.

          Example use should be the following:

          div {
          // regular styles here
          @include respondTo(desktop) {
          &:hover { background: blue; } // only add the hover effect on desktop browsers
          }
          }

                    David Tintner Bruno • 2 years ago

                    Hey bruno, The idea with $media = smallScreen was that I wanted a way to call styles that need to apply to anything up to the size of my normal desktop breakpoint. So it indeed is supposed to trigger for both tablet and mobile screen sizes.

                    However, if you need a media query for just smartphone or just tablet, you can use each of those breakpoints individually instead. If you want a media query for styles only on desktop, then use the 'desktop' breakpoint.

                    Also, you may very well want to adjust the pixel widths that I chose for breakpoints to match your project specifics.
