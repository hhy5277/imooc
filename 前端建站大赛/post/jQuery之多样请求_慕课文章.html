JQuery 1.9.0

普通请求

load() 异步请求数据

    $(selector).load(url,[data],[callback])
    语法
        selector :将从服务器上请求的数据,存储在selector元素中.

        url : 加载数据的地址

        data : 请求时发送的数据

        callback : 当加载完成时执行的回调函数

getJSON() 异步加载JSON数据

    $.getJSON(url,[data],[callback])等价
        于jQuery.getJSON(url,[data],[callback])

getScript() 异步加载并执行Js文件

    jQuery.getScript(url[,callback])等价
        于$.getScript(url[,callback])

    注意:在加载到本地时会执行该js代码段

GET POST请求

get() 方法

    采用GET方式向服务器请求数据,并通过回调函数的参数处理返回数据

    $.get(url,[callback])

    实例:

        $.get("SomeUrl",function(data){
        $("ul").append("<li>我是AJAX的"+data.name+"</li>")
        },"json")

    注意:

        在回调函数的右侧以带有引号json字符串
        指定返回数据格式
post() 方法

    $.post(url,[data],[callback])

    以post方式向服务器发送请求数据

    语法:

        data 是以JSON的形式发送数据

    实例:
        //相对路径下的php文件
        $.post("Data/check.php",{
            //获取本页面中id为txtNumber的值
            num:$("#txtNumber").val()
        },function(data){
            $("ul").append("<li>我是AJAX的"+data.name+"</li>")
        })

        注意:
            这是回调函数的右侧不再出现"json",来指定返回参数的格式

Ajax集成方法

使用ajax来发送和请求数据时,可以分为三个步骤:请求发送前,请求发送时,请求发送成功后.同时这三个阶段可激发三个函数.

并且还可以设置全局默认变量

请求发送前

$(selector).ajaxStart(function())

当发送ajax那一刻之前先会触发该函数

语法
    selector:选定的DOM元素

请求发送时

ajax() 加载服务器数据

    ajax()方法是最底层,功能最强大的请求服务器数据的方法,
            不仅仅可以获取服务器返回的数据,
            还能向服务器发送请求并传递数值.

    jQuery.ajax({settings})等价于$.ajax({settings})

    语法

        settings 为JSON对象

        url:      表示服务器请求的路径
        data:     请求时传递的数据
        dataType: 服务器返回的数据类型
        success:  请求成功时执行的回调函数
        type:     发送数据的格式,默认为get
实例
    $.ajax({
        url:"Data/article.txt",
        dataType:"text",
        success:function(data){
            //代码段
        },
        data:{num:$("#txtNumber").val()},
        type:"post"
    })

请求成功后

$(selector).ajaxStop(function())

当发送ajax请求结束后触发该函数

语法
    selector:选定的DOM元素

全局默认选项

ajaxSetup() 设置全局默认选项

    该方法可以设置Ajax请求的一些全局性选项值,设置成功后,
    后面的Ajax请求将不需要再添加这些选项值

    $.ajaxSetup([options])等价于$.ajaxSetup([options])

    实例:

        $.ajaxSetup({
            dataType:"text",
            success:function(){
                //代码段
            }
        })

知识加油站

serialize() 方法

    可以将表单中有name属性的元素值进行序列化,
        生成标准的URL编码文本字符串,直接可用于ajax请求

    $(selector).serialize()

    语法:

        selector参数是一个或多个表单中的元素或者表单元素本身

    注意:

        生成序列化的元素是表单中有name属性的元素
